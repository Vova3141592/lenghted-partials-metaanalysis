Финальный расчёт — в файле `metaanalysis.Rmd`. Это блокнот Markdown R, его можно открыть в R Studio и воспроизвести результат. Хотя я делал расчёты в JASP, этот код даёт идентичный результат, — только график менее красивый.

Данные для расчёта в файле `Метаанализ.csv`.

Ещё есть папка с исследованиями.

Для исследования Pedrosa et al. есть файл `wpd_project.json`, это проект для утилиты [WebPlotDigitizer](https://apps.automeris.io/wpd4/). Эта утилита нужна для извлечения данных с графика. После извлечения данных с графика нужно было дополнительные расчёты провести, они в файле `Дополнительные расчёты 2.xlsx`.
